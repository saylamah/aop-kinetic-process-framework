## Theory Background

Explains the core equations used in the framework, including:

* total scavenging capacity,
* apparent hydroxyl-radical concentration,
* apparent first-order rate constant,
* radical utilization efficiency,
* treatment-time estimation,
* simplified ozone and H2O2 radical-yield estimates.
