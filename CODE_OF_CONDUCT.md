# Code of Conduct

## Purpose

This repository is intended to support constructive scientific and engineering collaboration around advanced oxidation processes, wastewater treatment, and related process-analysis tools.

## Expected behavior

Participants are expected to:

- communicate respectfully,
- focus on technical and scientific content,
- provide constructive feedback,
- explain assumptions clearly,
- distinguish between evidence, interpretation, and opinion,
- respect different levels of expertise.

## Unacceptable behavior

Unacceptable behavior includes:

- personal attacks,
- harassment,
- discriminatory language,
- intentionally misleading technical claims,
- repeated off-topic comments,
- disrespectful or hostile communication.

## Scientific discussion

Technical disagreement is welcome when it is constructive.

When discussing equations, assumptions, data, or interpretations, contributors should aim to provide:

- clear reasoning,
- relevant references where possible,
- unit consistency,
- transparent assumptions,
- practical engineering context.

## Scope

This code of conduct applies to repository issues, pull requests, discussions, documentation, and related project communication.

## Reporting

If problematic behavior occurs, it can be reported to the repository maintainer.

The maintainer may remove inappropriate content, close issues, or restrict participation when necessary to preserve a constructive working environment.
