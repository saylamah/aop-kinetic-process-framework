# Contributing

Thank you for your interest in contributing to the AOP Kinetic Process Framework.

This repository is intended to provide transparent, scientifically conservative, and engineering-oriented Python tools for advanced oxidation process interpretation in wastewater treatment.

## Contribution priorities

Contributions are especially welcome in the following areas:

- correction of equations, units, or assumptions,
- additional examples,
- additional tests,
- improved documentation,
- validated rate-constant data,
- wastewater matrix datasets,
- literature benchmark cases,
- process-screening modules,
- uncertainty and sensitivity analysis,
- improved plotting and reporting tools.

## Scientific expectations

Contributions should be:

- transparent,
- well documented,
- unit-consistent,
- scientifically justified,
- conservative in interpretation,
- clear about assumptions and limitations.

Avoid adding black-box calculations without explanation.

## Data contributions

When adding rate constants, matrix values, or benchmark data, please include:

- units,
- source or reference,
- assumptions,
- applicable pH or temperature range if known,
- whether the value is measured, estimated, or representative.

## Code style

Please keep code:

- simple,
- readable,
- modular,
- tested,
- documented with docstrings.
